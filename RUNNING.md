# Quick run instructions

1. Create a Telegram bot with @BotFather and obtain the token.
2. Copy `.env.example` to `.env` and edit with your token and Telegram user id.
3. Install dependencies: `pip install -r requirements.txt`.
4. Initialize the database: `python database/init_db.py`.
5. Start the bot: `python main.py`.
6. Open Telegram, send /start to your bot and press "Start AutoTrader".
